创建博客的一些感想

先吐槽吧

说实话真心不知道博客这个东西有个什么用，可能我百度了这么多年偶尔进过一两个人的博客，再然后就是查和博客相关的内容的时候进去过（算自娱自乐吗），但是没啥感觉，get自己要的东西就吧网页叉掉了，可能我不适合做个程序员吧。而且在换主题的时候，我浏览了一大半hexo的主题（另一半要么没demo、要么404、还有的打不开），不得不说大部分程序员喜欢的简洁真心难看啊（又一个不适合做程序员的理由→_→），成天嚷嚷着要什么代码就够了，图形化不需要的，我才不想和他们一样，虽然我品味估计也有问题，但是我还是想坚持我喜欢的，然而稍微喜欢的的h5的主题根本没多少能改的，还得学css、js什么鬼的去更深处改，真的迷……恩主题还得慢慢改…还有就是写博客，markdown是真的迷啊，段首空格就成换行了，还得学新语法，语法是迷上加迷，说是兼容h5语法，开玩乐呢吧，我用h5的a标签超链接都不行（我就是懒，不想学新语法想麻烦你奈我何），有空再学了，先打两把dead game冷静下……


再说说根据教程搭建博客遇到的坑吧

我是根据这个 http://www.jianshu.com/p/2b9f202c13fd 教程来的，首先得感谢下这个作者，他说他用了三天搭建，我比较笨，根据他的教程还用了一天……不过其中有个锅估计得我自己背，就是这里

四、部署到GitHub

配置_config.yml

编辑E:\hexo 下的_config.yml，修改 Deployment部分

Deployment
Docs: https://hexo.io/docs/deployment.html
ssh://git@github.com/git-xuhao/git-xuhao.github.io
deploy:
type: git
repo: ssh://git@github.com/git-xuhao/git-xuhao.github.io
branch: master

就这个repo，我除了把我的ssh复制过来了，还把他开头的ssh://给补上去了，所以可想而知，我根本打不开这个网页，gg思密达后我就从头再来重复了4遍，4遍啊老铁，心态都崩了，只能说输一手我笨……恩，最后突然灵光一闪觉得好像这里有可坑，结果还真有可坑……真的惨。


最后说说我对我的博客的想法

emmmm…….我不大喜欢博客这个东西，主要是不知道博客有什么用处而且对于现在的我来说写起来太麻烦，我作为一个菜鸡不像别的大佬有些有用的东西可写，而且即使没人看我也不会把一些涉及到隐私的心理活动写到这里，虽然我希望别人能了解我，但重要的东西还是得放在重要的地方给别人看，而不是这里。所以除了被强迫来写东西我怕是不会来了，可能偶尔来放点游戏的通关感想吧，因为我这人除了游戏和推理电影没啥特殊的爱好了估计，剩下的兴趣爱好还在开发中…
